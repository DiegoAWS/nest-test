## Deployed Project

To see this project you could visit <https://nest-test-backend.herokuapp.com/> or you could clone it and run it locally.

## Installation

```sh
npm install
```

## Running the app

```sh
# development
npm run start

# watch mode
npm run start:dev

# production mode
npm run start:prod
```

## Test

```sh
# unit tests
npm run test

# e2e tests
npm run test:e2e

# test coverage
npm run test:cov
```
